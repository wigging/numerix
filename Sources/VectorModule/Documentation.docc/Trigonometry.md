# Trigonometry

Trigonometic vector operations.

## Overview

Numerix provides several element-wise trigonometic functions for vectors. The Accelerate framework is utilized for efficient element-by-element vector operations. 

## Topics

- ``sin(_:)``
- ``cos(_:)``
- ``tan(_:)``
- ``asin(_:)``
- ``acos(_:)``
- ``atan(_:)``
- ``csc(_:)``
- ``sec(_:)``
- ``cot(_:)``
